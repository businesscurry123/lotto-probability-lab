# Google Play 스토어 등록 문구 최종안

## 앱 이름

로또 누적확률 연구소 (AI)

## 짧은 설명

저장한 로또 번호의 적중 기록, 과거 통계, 누적확률 계산을 확인하는 참고 도구입니다.

## 자세한 설명

로또 누적확률 연구소 (AI)는 사용자가 저장한 로또 번호를 기준으로 회차별 기록, 과거 통계, 누적 시행 계산을 보기 쉽게 확인하는 참고 도구입니다.

주요 기능:

- 이번 회차 당첨번호와 과거 회차 결과 확인
- 저장한 번호 세트와 즐겨찾기 관리
- 저장 번호 기준 적중 기록과 성장 기록 확인
- 내 저장 번호끼리 이번 회차 결과 비교
- 아깝다 지수와 근접 기록 확인
- 번호별 과거 출현 통계와 미출현 기간 확인
- 누적 시행 확률 계산기
- 회차별 분석 리포트
- 앱 내부 확인용 QR 표시
- 로컬 알림 설정

안내:

이 앱은 번호를 예측하거나 당첨 확률을 높인다고 안내하지 않습니다.
모든 통계와 누적확률 계산은 과거 기록과 수학적 계산을 보기 쉽게 보여주는 참고 지표입니다.
다음 회차의 실제 확률은 모든 번호와 조합에 동일하게 적용됩니다.

QR 기능은 앱 내부 확인용이며, 공식 모바일 슬립 연동 기능을 제공하지 않습니다.

## 카테고리

- 앱 카테고리: 도구
- 태그 후보: 통계, 계산기, 기록

## 광고 포함 여부

- 광고 포함: 예

Google Mobile Ads SDK를 사용하므로 Play Console의 광고 포함 여부는 `예`로 신고합니다.

## Play Console 연락처

- 이메일: Play Console 개발자 계정 이메일 또는 앱 문의용 이메일
- 웹사이트: 선택 사항
- 전화번호: 선택 사항

## 문구 작성 주의사항

스토어 등록정보, 앱 홍보 문구, 스크린샷 설명에서는 아래 방향의 표현을 사용하지 않습니다.

- 특정 번호가 더 유리하다고 말하는 표현
- AI가 결과를 맞힌다고 보이게 하는 표현
- 다음 회차 결과를 미리 안다고 보이게 하는 표현
- 특정 번호의 실제 확률이 개선된다고 보이게 하는 표현
- 공식 서비스와 연결된 것처럼 보이는 표현

대신 아래 표현을 사용합니다.

- 과거 기록 기준 통계
- 누적 시행 확률 계산
- 번호 구성 참고 지표
- 앱 내부 확인용 QR
- 미래 출현을 보장하지 않습니다.
