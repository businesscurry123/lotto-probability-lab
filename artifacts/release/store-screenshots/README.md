# Play Console 스토어 스크린샷 후보

아래 6장을 우선 업로드 후보로 사용합니다.

1. `store-01-home.png` - 홈 화면
2. `store-02-feature-center.png` - 부가기능 센터
3. `store-03-save-qr-settings.png` - 번호 저장 / QR 설정
4. `store-04-number-stats.png` - 번호 통계
5. `store-05-probability-warning.png` - 누적확률 경고 화면
6. `store-06-my-analysis.png` - 내 번호 분석

각 이미지는 Android 휴대전화 스크린샷 비율이며, Play Console 업로드 전 실제 화면 문구가 최신 앱과 맞는지 한 번 더 확인합니다.
